# 📊 EduPro Online Learning Analytics Project

## 📌 Project Overview

This project analyzes learner demographics and course enrollment behavior on the EduPro online learning platform.
The goal is to understand user engagement, course preferences, and demographic patterns to improve learning strategies and business decisions.

## 🎯 Problem Statement

Despite having user and transaction data, EduPro lacks clear insights into:

* Which age groups are most active on the platform
* How enrollment patterns differ by gender
* Which course categories are most preferred
* Whether beginners, intermediate, or advanced courses are more popular among different age groups

## 📂 Dataset Description

The dataset includes:

* Users data (User demographics like age, gender)
* Courses data (Course category, level, type)
* Transactions data (Enrollment records)

Final merged dataset is used for analysis.

## ⚙️ Tools & Technologies Used

* Python (Pandas, NumPy)
* Matplotlib & Seaborn
* Google Colab
* Power BI
* Streamlit (optional dashboard)

## 🧹 Data Processing Steps

1. Data Loading from CSV files
2. Data Cleaning (handling missing values & duplicates)
3. Data Merging (Users + Courses + Transactions)
4. Feature Engineering (Age Group creation)
5. Exploratory Data Analysis (EDA)

## 📊 Key Analytical Questions Answered

* Which age group is most active on EduPro?
* How do enrollment patterns vary by gender?
* Which course categories are most popular?
* Are beginner or advanced courses more preferred?
* How do different age groups behave in course selection?

## 📈 Key Insights

* Most learners belong to the 26–35 age group
* Beginner courses have the highest enrollments
* Technology-related courses are most popular
* Gender distribution is nearly balanced
* Course preference varies across age groups

## 📊 Dashboard

A Power BI dashboard was created including:

* KPI Cards (Total Learners, Courses, Enrollments)
* Age Distribution Analysis
* Gender Distribution
* Course Category Analysis
* Course Level Analysis
* Interactive filters (Slicers)

## 📌 KPIs Calculated

* Total Learners
* Total Courses
* Total Enrollments
* Most Popular Course Category
* Most Popular Course Level


## 🚀 Conclusion

This project provides meaningful insights into learner behavior on EduPro.
It helps in designing better learning experiences and improving course recommendations based on data-driven decisions.
